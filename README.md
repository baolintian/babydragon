# babydragon

Minimal Hugo blog scaffold for a Yinwang-inspired reading-focused site.

## Bootstrap check

Before `hugo.toml` exists, `hugo --gc --minify` is expected to fail with a missing configuration error. That failing run is the initial configuration check for this project bootstrap task.

## Development

Run `hugo server` for local preview.

## Build

Run `hugo --gc --minify` to generate the static site into `public/`.
